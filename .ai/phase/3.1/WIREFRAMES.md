# Phase 3.1: OpenRouter Client - Wireframes

**Phase:** 3.1
**Name:** OpenRouter Client
**Dependencies:** Phase 1.1 (Project Foundation), Phase 1.2 (Configuration)

---

## Overview

Phase 3.1 is primarily a backend/API integration phase. The wireframes focus on:
1. Data structure formats
2. API request/response examples
3. Error message formats
4. Debug/logging output

---

## 1. API Request Formats

### Basic Chat Completion Request
```json
{
  "model": "anthropic/claude-3-opus",
  "messages": [
    {
      "role": "system",
      "content": "You are a helpful coding assistant."
    },
    {
      "role": "user",
      "content": "How do I read a file in Python?"
    }
  ],
  "temperature": 0.7,
  "max_tokens": 1000
}
```

### Request with Tools
```json
{
  "model": "anthropic/claude-3-opus",
  "messages": [
    {
      "role": "user",
      "content": "Read the file at /home/user/test.py"
    }
  ],
  "tools": [
    {
      "type": "function",
      "function": {
        "name": "Read",
        "description": "Read contents of a file",
        "parameters": {
          "type": "object",
          "properties": {
            "file_path": {
              "type": "string",
              "description": "Absolute path to the file"
            }
          },
          "required": ["file_path"]
        }
      }
    }
  ],
  "tool_choice": "auto"
}
```

### Request with Routing Variant
```json
{
  "model": "anthropic/claude-3-opus:nitro",
  "messages": [...],
  "temperature": 0.7
}
```

### Streaming Request
```json
{
  "model": "anthropic/claude-3-opus",
  "messages": [...],
  "stream": true
}
```

## 3. Error Response Formats

### Authentication Error (401)
```json
{
  "error": {
    "message": "Invalid API key provided",
    "type": "authentication_error",
    "code": "invalid_api_key"
  }
}
```

### Rate Limit Error (429)
```json
{
  "error": {
    "message": "Rate limit exceeded. Please retry after 30 seconds.",
    "type": "rate_limit_error",
    "code": "rate_limit_exceeded"
  }
}
```

Headers:
```
Retry-After: 30
```

### Model Not Found (404)
```json
{
  "error": {
    "message": "Model 'fake/model' not found",
    "type": "not_found_error",
    "code": "model_not_found"
  }
}
```

### Context Length Error (400)
```json
{
  "error": {
    "message": "This model's maximum context length is 128000 tokens. You requested 150000 tokens.",
    "type": "invalid_request_error",
    "code": "context_length_exceeded"
  }
}
```

### Content Policy Error (400)
```json
{
  "error": {
    "message": "Content violates usage policy",
    "type": "invalid_request_error",
    "code": "content_policy_violation"
  }
}
```
